zoomed-in plans


new code regions:
    mourningwail:
        new django app for metrics/analytics
    mourningwail.metrics.events:
        models for storing event-stream data in elasticsearch
    mourningwail.metrics.reports:
        models for storing report data in elasticsearch
    mourningwail.reporters: 
        given a date, a reporter generates data about what happened on that date
        `reporter.report(date)` returns iterable of `mourningwail.metrics._base.MeteredReport` instances


constrained query endpoint:
    SUM of integer field over bucketed time interval
    COUNT on keyword field within given time interval
    decouple api field names from elasticsearch field names


new api surface:
    /_/metrics/node_analytics/<guid>/<timespan>/ (GET everything needed for the node analytics page)
    /_/page_visit/ (POST new page-visit events)
